##################################################
Description of input files in Input/Biodiveristy
##################################################

1) Independent variables used in the SDMs

- Bio_1.tif, Data source:(Fick & Hijmans, 2017; Harris et al., 2020)

- Bio_4.tif, Data source: (Fick & Hijmans, 2017; Harris et al., 2020)

- Bio_12.tif, Data source: (Fick & Hijmans, 2017; Harris et al., 2020)

- Bio_15.tif, Data source: (Fick & Hijmans, 2017; Harris et al., 2020)

- Elevation.tif, Data source: (EROS, 2017)

- Slope.tif, Data source: (EROS, 2017)



2) Coeficient and performance indicators of the SDMs (Generated using InputBiodiveristy3.R, stored in "Input/Premilinary_input/Biodiveristy")

- Species_coef.xlsx

- Performance.xlsx


3) Species ranges in tiff are stored in Biodiversity/Species_Range directory

Data source: (Marsh et al., 2022) 


4) Other input

- Ott_order.xlsx

- Range_Ott_id.xlsx
