##################################################
Input to generate the reference Carbon Stock values
##################################################


Climate zone: Climate_IPCC  (IPCC, 2006; IPCC, 2019)

Stock Factor Changes: Fc.Xlsx  (MCTI, 2020; SIRENE et al., 2020, IPCC, 2006; IPCC, 2019). 

Reference value for Soil Organic Carbon: SOCref_subordem.xlsx   (MapBiomas Project, 2023a)

Reference value for Biomass Carbon: MCTI_ref.xlsx & IPCC_ref.xlsx   (MCTI, 2020; SIRENE et al., 2020, IPCC, 2006; IPCC, 2019). 

Other input data:

	- Soil type and climate type for each cell: ID_CLimSoil_Subordem.xlsx  (IBGE, 2024, Alvares et al., 2013)

	- Biome and  Phytoregion  for each cell: ID_BiomePhyto.xlsx  (MCTI, 2020; IBGE, 2024) 

	- File to desagragate the Mosaic and croplands: Percentage_agri_mos.xlsx  (MapBiomas Project, 2023b; IBGE, 2022)
