########################################################
Input directory
########################################################


Contain all the necceassary input data to run the script in "Create_input.py", including:

	- Input data to generate the data to calculate the change in carbon stock are stored in "Input/Preliminary_Input/Carbon" 
	
	- Input data to generate the data to calculate the change in Agricultural revenue are stored in "Input/Preliminary_Input/Economy" 
	
	- Original land use projection, before desagragation of cropland and Mosaic are stored in "Input/Preliminary_Input/Land_Use" 
	  Data source: Silva Bezerra, F. G., Von Randow, C., Assis, T. O., Bezerra, K. R. A., Tejada, G., Castro, A. A., Gomes, D. M. de P., 
          Avancini, R., & Aguiar, A. P. (2022). New land-use change scenarios for Brazil: Refining global SSPs with a regional spatially-explicit 
          allocation model. PLOS ONE, 17(4), e0256052. https://doi.org/10.1371/journal.pone.0256052
	
	- Script to generate the data to calculate the change in species richness are stored in "Input/Preliminary_Input/Biodiveristy"
